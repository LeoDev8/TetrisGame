# Tetris Game

## Installation

```shell
git clone https://github.com/LeoDev8/TetrisGame.git
```

```shell
cd TetrisGame
```

```shell
npm start
```

```shell
npm run ios-15pm
```



## Publish Issues

```bash
brew install gh
```

```shell
gh auth login
```

```shell
gh issue create --title "Issue Title" --body "Issue Description"
```
